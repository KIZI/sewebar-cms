There are directories containing different types of tests.

SWBRTests
Tests from Tomáš Kliegr. These tests contains sample ARQuery

FirebugTests
Tests that need firebug console to be started. Therefore they can be replayed in
either Firefox or Google chrome.

testSelenium
Selenium tests, which need Selenium IDE for running them.

testSerializeRules
Specific test, testing serializing JSON to three formats ARQuery, TaskSetting
and AsociationRule
