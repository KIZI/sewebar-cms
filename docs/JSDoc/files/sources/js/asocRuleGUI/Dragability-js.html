Dragability

This class ensures Dragability on choosen elements.

Summary
DragabilityThis class ensures Dragability on choosen elements.
Functions
initializeIt creates instance of class Dragability
dropFunctionIt solves whether the element is dropped on correct place
completeFunctionThis function is called when the element is dropped.
snapFunctionIt creates a copy of element that is being dragged.
createDragabilityThis function is called quite often and it adds to all elements, that should be draggable, dragability.
removeDragabilityIt is necessary to remove ability to be draged and to be draged to in order to achieve ability to be draged to on new places.

completeFunction

completeFunction: function(element)

This function is called when the element is dropped.  It looks if the element is on the right place and if it isn’t.  It is removed.

Parameters

element {HTMLElement} dropped element.

snapFunction

snapFunction: function(element)

It creates a copy of element that is being dragged.  And say that there should be no more copies of this element.

Parameters

element {HTMLElement} The element that should be copied.

createDragability

createDragability: function()

This function is called quite often and it adds to all elements, that should be draggable, dragability.  It also makes each of the places to be able to receive tha draggable element.

removeDragability

removeDragability: function()

It is necessary to remove ability to be draged and to be draged to in order to achieve ability to be draged to on new places.
