Requirements 
These tests need Firebug console.

Running:
Open any of files with name.html. Results are in Firebug console. If you can't
see any the try reloading page.

testAsociationRule.html
Tests whether class AssociationRule works correctly. It tests inserting item,
removing item, changing item and some of inner functions which are not exposed
publicly

testControls.html
Test whether class Controls works correctly. Ir provides functionality for testing
if value is of corect type and if it is in allowed boundaries.

testDepthNesting.html
Test correct behavior of class DepthNesting. Basically it tests whether DepthNesting
allows correct connectives on correct levels.
